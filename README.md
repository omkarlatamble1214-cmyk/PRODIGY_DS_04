# PRODIGY_DS_04

## Task 04: Sentiment Analysis

### Objective

Analyze and visualize sentiment patterns in social media data to understand public opinion and attitudes toward specific topics or brands.

### Tools Used

* Python
* Pandas
* TextBlob
* Matplotlib
* Google Colab

### Process

1. Created a dataset of social media posts.
2. Performed sentiment analysis using TextBlob.
3. Classified sentiments as Positive, Negative, or Neutral.
4. Visualized sentiment distribution using a bar chart.

### Results

* Identified positive and negative opinions in the dataset.
* Visualized overall sentiment distribution.

### Files Included

* PRODIGY_DS_04.ipynb
* sentiment_distribution.png
* README.md

### Conclusion

Sentiment analysis helps understand public opinion and customer feedback by categorizing text into sentiment groups.
